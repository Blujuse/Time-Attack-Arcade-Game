# Time-Attack-Arcade-Game
 
This is my own repository for the individual arcade game, this does not contain the team game

Features:
* Race your car downhill!
* Use the momentum to carry yourself and drift the corners!
* How fast can you be?

## Dependencies
* Built in Unity
